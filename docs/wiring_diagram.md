# 결선 매뉴얼 — 자율수거 로봇 시제품

조립 시 헷갈리지 않도록 **모든 점-대-점 연결**을 정리. `arduino_firmware/config.h`와 일치.

---

## 0. 표기 약속 (전선 색상)

| 색상 | 의미 | 어디에 |
|------|------|--------|
| 🔴 **빨강** | 7.4V 배터리 라인 | LiPo (+) → 분배 → 컨버터 입력, L298N VS |
| 🟠 **주황** | 5V 로직 (LM2596HV 출력) | Arduino, 빵판 5V, L298N 5V, 서보 (+) |
| 🟡 **노랑** | 5V RPi (XL4015 출력) | RPi USB-C 입력 전용 |
| ⚫ **검정** | GND (모든 접지) | 스타 접지 허브 |
| 🔵 **파랑** | 디지털 신호 | Trig/Echo, IN1~4, 서보 신호 |
| 🟢 **초록** | I2C / UART | SDA, SCL |
| 🟣 **보라** | PWM | ENA/ENB |

테프론선이 빨강/검정 1m씩만 있으니 **굵은 트렁크 라인(배터리→분배)** 에만 사용. 분배 후엔 일반 점퍼선 OK.

## 1. 전원 분배 트리 (듀얼 배터리: 로직 7.4V + 모터 12V)

### 1-1. 로직 배터리 (7.4V LiPo) — Layer 2 좌측
```
[7.4V LiPo (+)]
   │ 🔴 (테프론 빨강)
   │
   ├─[로직 메인 스위치]
   ├─[5A 퓨즈]
   └─[1000µF 전해 캡]──┐
                       │
   ┌───────────────────┘  (2P 터미널 블록)
   │
   ├─🔴→ XL4015 #1 IN+ →(출력 5.10V)→ 🟡 RPi USB-C
   │                                       │
   │                                       └→ USB-A → Arduino (전원 + 데이터)
   │
   └─🔴→ LM2596HV IN+  →(출력 5.00V)→ 🟠 빵판 5V 레인
                                            ├→ HC-SR04 ×5
                                            ├→ MPU-9250 (+ 4.7kΩ 풀업)
                                            ├→ Arduino 5V 핀 (보조 급전)
                                            └→ MG996R 서보 (+) 굵은 별도선

[7.4V LiPo (-)] ─⚫─ ★ GND 버스바
```

### 1-2. 모터 배터리 (12V LiPo) — Layer 2 우측
```
[12V LiPo (+)]
   │ 🔴 (테프론 빨강 굵게, 18AWG 권장)
   │
   ├─[모터 메인 스위치]
   ├─[10A 퓨즈]
   ├─[비상정지 버튼 NC]   ← 사람이 누르면 모터만 차단 (RPi/센서는 살아있음)
   └─[2200µF 전해 캡]──┐
                       │
   ┌───────────────────┘
   │
   └─🔴→ XL4015 #2 IN+
            ⚠️ 출력을 7.4V로 사전 조정 (멀티미터 필수)
            (12V 그대로 L298N에 보내면 모터에 ~10.5V → 6V 모터 소손!)
            │
            ├─🔴→ L298N #1 VS → NP01D-288 ×2 (병렬, 후륜)
            └─🔴→ L298N #2 VS → 롤러 모터 ×2

[12V LiPo (-)] ─⚫─ ★ GND 버스바 (로직 배터리와 같은 버스바!)
```

### 1-3. 공통 GND 버스바 (스타 접지)
```
★ GND 버스바 ←──────┐
   ├─⚫─ 7.4V LiPo (-)
   ├─⚫─ 12V LiPo (-)        ← 두 배터리 GND가 여기서 만남 (공통 기준)
   ├─⚫─ XL4015 #1 IN-
   ├─⚫─ XL4015 #2 IN-
   ├─⚫─ LM2596HV IN-
   ├─⚫─ L298N #1 GND
   ├─⚫─ L298N #2 GND
   ├─⚫─ Arduino GND
   ├─⚫─ RPi GND
   └─⚫─ 빵판 GND 레인 ─────┘
```

⚠️ **두 배터리의 GND가 반드시 한 점(버스바)에서 만나야** 함. 안 그러면 통신 신호가 기준 전위를 잃음.
⚠️ **GND를 절대 직렬(daisy chain)로 연결하지 말 것**. 모터 GND가 Arduino GND를 거쳐 흐르면 신호 흔들림.

### 1-4. XL4015 #2 출력 전압 설정 가이드
| 모터 정격 | XL4015 #2 출력 | L298N 강하 후 모터 도달 |
|----------|---------------|----------------------|
| 6V (NP01D-288) | **7.4V** ✅ | ~5.9V (정격) |
| 6V (안전 마진) | 6.5V | ~5.0V (토크 ↓, 안전) |
| 12V 모터로 변경 시 | 12V 직결 | ~10.5V (정격 OK) |

→ **현재 6V 모터 사용 → XL4015 #2를 7.4V로 미리 조정**.

## 2. RPi 4 (Layer 4)

| RPi 핀/포트 | 연결 대상 | 색 | 비고 |
|-------------|----------|----|------|
| **USB-C 입력** | XL4015 5V 출력 (5.10V로 사전 조정 필수) | 🟡 + ⚫ | 멀티미터로 확인 후 연결 |
| **USB-A 포트 (파란색 USB 3.0 권장)** | Arduino USB-B 단자 | — | 데이터 전용. **VBUS 차단 USB 케이블** 사용 권장 (이중 5V 공급 방지) |
| **CSI 카메라 포트** | RPi 카메라 모듈 3 | (전용 리본 케이블) | 검은 잠금 클립 들어올린 후 삽입, 클립 닫기 |
| **USB-A 포트 (검은 USB 2.0)** | 웹캠 AU100 | (USB 케이블 자체) | — |
| GPIO 6번/9번 (GND) | (옵션) GND 버스바 | ⚫ | USB-C로 GND 들어가지만 추가 GND 연결 권장 (안정성) |

> 💡 USB 케이블 VBUS 차단법: 빨간 선만 잘라내면 됨 (D+/D-/GND 살림). 또는 시판 "데이터 전용 USB 케이블" 구매.

## 4. L298N #1 — 구동 모터 (Layer 2)

### 4-1. 점퍼 제거 ⚠️
구매 후 **5V 활성화 점퍼(작은 노란 캡)를 빼서 보관**. 외부 5V 모드로 전환됨.

### 4-2. 결선
| L298N #1 핀 | 연결 | 색 |
|-------------|------|----|
| **+12V (또는 VS)** | LiPo (+) 7.4V (분배 터미널) | 🔴 |
| **GND** (3개 모두) | GND 버스바 | ⚫ |
| **+5V** | LM2596HV 5V (점퍼 제거 후 외부 공급) | 🟠 |
| **OUT1** | 좌측 후륜 모터 (+) | 🔴 (모터선 빨강) |
| **OUT2** | 좌측 후륜 모터 (-) | ⚫ (모터선 검정) |
| **OUT3** | 우측 후륜 모터 (+) **— ⚠️ OUT1과 병렬 결선** | 🔴 |
| **OUT4** | 우측 후륜 모터 (-) **— ⚠️ OUT2와 병렬 결선** | ⚫ |
| **ENA** | Arduino 핀 5 | 🟣 |
| **IN1** | Arduino 핀 32 | 🔵 |
| **IN2** | Arduino 핀 33 | 🔵 |
| ~~ENB, IN3, IN4~~ | (사용 안 함, 비워둠) | — |

> ⚠️ **두 후륜을 한 채널에 병렬 연결**: OUT1/OUT3을 점프 연결, OUT2/OUT4도 점프 연결. 이렇게 두 모터가 같은 PWM/방향으로 동시 회전.

### 4-3. 노이즈 대응
- 모터 단자에 **0.1µF 세라믹 캡** ×2 (모터 +/- 단자 양단)
- L298N VS↔GND에 **1000µF 전해 캡** (극성 주의: + 다리 → VS)
- 모터 배선에 **페라이트 코어** 한 번 감기

## 10. 통전 검증 순서 (조립 후)

배터리 연결 **전에** 멀티미터로 단락 체크 → 배터리 연결 → 단계별 확인.

### Step 1: 배터리 연결 안 한 상태
- LiPo (+) 라인과 GND 버스바 간 저항: **>1kΩ** (단락 X 확인)
- 모든 모터 (+/-) 단자가 GND 버스바와 절연 (∞)

### Step 2: 배터리 연결 + 메인 스위치 ON (모터 X)
- 멀티미터로 측정 (먼저 **로직 배터리 7.4V만** 연결):
  - 7.4V LiPo 직후: **7.4V ~ 8.4V**
  - XL4015 #1 출력: **5.10V ± 0.05**
  - LM2596HV 출력: **5.00V ± 0.05**
  - Arduino 5V 핀: **5.0V**
- RPi 부팅 LED 확인 (4초 내 빨간/초록 LED 점등)
- Arduino LED 점등 확인 (PWR + 13번 핀 깜빡)
- 시리얼 모니터로 부팅 메시지 확인

- 다음 **모터 배터리 12V** 연결:
  - 12V LiPo 직후: **12V ~ 12.6V**
  - **XL4015 #2 출력: 7.4V ± 0.1** ⚠️ (반드시 확인! 12V 그대로면 모터 소손)
  - L298N #1, #2 VS: **7.4V** (XL4015 #2와 같음)
  - L298N 5V 핀: **5.0V** (LM2596에서 들어온 외부 5V)

### Step 3: 시리얼 모니터 확인
```bash
# Arduino IDE 시리얼 모니터 (115200, Newline)
# 또는: python -m tools.telemetry_monitor
```
부팅 메시지 확인:
```json
{"event":"boot","imu":true}
```
- `"imu":true` → MPU-9250 정상
- `"imu":false` → I2C 풀업 저항 또는 SDA/SCL 결선 확인

### Step 4: 센서 검증 (`telemetry_monitor.py`)
- 손을 전방 센서 가까이 → 거리 줄어듦 ✅
- 로봇 회전 → yaw 변화 ✅
- 기울임 → pitch/roll 변화 ✅

### Step 5: 모터 검증 (`manual_control.py`, 차체 들어올린 상태)
- W 누름 → 후륜 둘 다 같은 방향 회전 ✅
  - 거꾸로 돌면 → IN1↔IN2 핀 스왑 또는 모터 +/- 스왑
- A/D 누름 → 서보 좌/우 회전 ✅
  - 반대로 가면 `STEER_RANGE_DEG` 부호 반전 또는 서보 마운트 뒤집기
- R 누름 → 롤러 ×2 회전 ✅

## 11. 흔한 함정 (failure 케이스)

| 증상 | 원인 | 해결 |
|------|------|------|
| RPi가 갑자기 리부팅 | 모터 기동 시 7.4V 새그 | 2200µF 캡 추가, 모터 가속 ramp-up |
| Arduino 부팅 메시지 무한 반복 | USB로 양쪽에서 5V 들어옴 | USB 케이블 VBUS 차단 |
| `"imu":false` | I2C 풀업 저항 누락 또는 SDA/SCL 바뀜 | 4.7kΩ ×2 빵판에 추가 |
| HC-SR04 모두 0xFFFF (∞) | VCC 5V 미공급 또는 GND 분리 | 빵판 5V/GND 레인 확인 |
| 한 모터만 안 돔 | OUT1↔OUT3 병렬 점프선 끊김 | 점프선 재연결 |
| 모터가 PWM 무관하게 풀스피드 | L298N 5V 점퍼 안 빠짐 → IN 핀 떠다님 | 점퍼 제거 + 외부 5V 공급 |
| 서보가 떨림 | 서보 (+) 가 Arduino 5V 핀에 연결됨 | LM2596 직결로 변경 |
| 텔레메트리 끊김 → safe=false → 정지 | RPi 부팅 30초간 명령 안 보냄 | 정상. RPi 준비될 때까지 대기 |

## 12. 한 페이지 요약 카드 (조립 시 옆에 두기)

```
[7.4V LiPo (+)]──[스위치]──[퓨즈 5A]──┬──XL4015 #1 ──5.1V── RPi USB-C ── USB ── Arduino
                                       └──LM2596HV  ──5.0V── 빵판 5V
                                                              ├─ HC-SR04 ×5 (VCC)
                                                              ├─ MPU-9250 (+ 4.7kΩ ×2 풀업!)
                                                              ├─ Arduino 5V (보조)
                                                              ├─ MG996R (+) 별도 굵은선
                                                              └─ L298N #1, #2의 5V 핀

[12V LiPo (+)]──[스위치]──[퓨즈 10A]──[E-stop]──┬──XL4015 #2 ──7.4V── L298N #1 VS ── 후륜 ×2 (병렬)
                                                                       └─ L298N #2 VS ── 롤러 ×2

[7.4V (-)] ┐
[12V (-)]  ├──★ GND 버스바 ★── 모든 GND (각자 별도선)
모든 GND ─┘

저항: MPU-9250 I2C 풀업용 4.7kΩ ×2 (이게 전부!)

핵심 체크리스트:
• L298N #1, #2의 5V 점퍼 제거
• XL4015 #2 출력을 멀티미터로 7.4V 확인 (12V 그대로면 모터 소손!)
• MPU-9250 SDA/SCL에 4.7kΩ 풀업 ×2
• 두 배터리 GND가 같은 버스바에서 만남 (스타 접지)
• 모든 GND는 별도선으로 버스바에 직결 (직렬 X)
• USB 케이블 VBUS 차단 (RPi-Arduino 이중 5V 방지)
• 배터리 연결 전 단락 체크 (멀티미터 >1kΩ)
```
